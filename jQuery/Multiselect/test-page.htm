Multiselect jQuery widget

With this small jQuery widget, you can extent the autocomplete functionalities. It is possible to convert a <select> tag to a multi-autocomplete. But this can also be added to a simple <input> tag.

Example with <select>

<div>
	<select id="AutocompleteSelect">
		<option value="1">Car</option>
		<option value="2">Bike</option>
		<option value="3">Ferry</option>
		<option value="4">Plane</option>
		<option value="5">Monowheel</option>
	</select>
</div>

<script type="text/javascript">$("#AutocompleteSelect").autocomplete();</script>

Example with <input>

<input id="AutocompleteInput">

<script type="text/javascript">
	$("#AutocompleteInput").autocomplete({
		multiselect: true,
		source: [ 
			{ value: "1", label: "Car" }, 
			{ value: "2", label: "Bike" }, 
			{ value: "3", label: "Ferry" }, 
			{ value: "4", label: "Plane" }, 
			{ value: "5", label: "Monowheel" }
		]			
	});
</script>
